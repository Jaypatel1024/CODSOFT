# Simple ATM System

This is a simple Java program that simulates an ATM system. It allows users to perform basic banking operations such as withdrawing money, depositing money, and checking their account balance.

## Features

- Withdraw money: Users can withdraw money from their bank account. If the withdrawal amount exceeds the available balance, an appropriate message is displayed.
- Deposit money: Users can deposit money into their bank account. The deposited amount is added to the account balance.
- Check account balance: Users can check their account balance at any time.
- User input validation: The program validates user input to ensure that withdrawal and deposit amounts are positive values.
- Simple user interface: The program provides a simple command-line interface for users to interact with the ATM system.

## How to Use

1. Compile the Java files using a Java compiler.
2. Run the `Main` class to start the program.
3. Follow the on-screen instructions to navigate through the ATM menu and perform banking operations.

## Requirements

- Java Development Kit (JDK) installed on your system.

## Usage Example

$ javac Main.java
$ java Main
